# 3.　開発 - 入力フォーム画面を作る

* 以下ファイルに、UI入力フォームの雛形を記述します。
kotlin/org/example/project/App.kt

```Kotlin
package org.example.project

import androidx.compose.foundation.layout.*
import androidx.compose.material3.*
import androidx.compose.runtime.*
import androidx.compose.ui.Alignment
import androidx.compose.ui.Modifier
import androidx.compose.ui.unit.dp
import androidx.compose.ui.tooling.preview.Preview

@Composable
@Preview
fun App() {
    MaterialTheme {

        var name by remember { mutableStateOf("") }
        var bio by remember { mutableStateOf("") }
        var github by remember { mutableStateOf("") }
        var zenn by remember { mutableStateOf("") }

        Column(
            modifier = Modifier
                .fillMaxSize()
                .padding(16.dp),
            horizontalAlignment = Alignment.CenterHorizontally
        ) {

            Text("ポートフォリオ自動生成", style = MaterialTheme.typography.headlineMedium)

            Spacer(Modifier.height(16.dp))

            TextField(
                value = name,
                onValueChange = { name = it },
                label = { Text("名前") },
                modifier = Modifier.fillMaxWidth()
            )

            Spacer(Modifier.height(8.dp))

            TextField(
                value = bio,
                onValueChange = { bio = it },
                label = { Text("自己紹介") },
                modifier = Modifier.fillMaxWidth()
            )

            Spacer(Modifier.height(8.dp))

            TextField(
                value = github,
                onValueChange = { github = it },
                label = { Text("GitHub URL") },
                modifier = Modifier.fillMaxWidth()
            )

            Spacer(Modifier.height(8.dp))

            TextField(
                value = zenn,
                onValueChange = { zenn = it },
                label = { Text("Zenn URL") },
                modifier = Modifier.fillMaxWidth()
            )

            Spacer(Modifier.height(16.dp))

            Button(onClick = {
                println("=== 入力データ ===")
                println("名前: $name")
                println("自己紹介: $bio")
                println("GitHub: $github")
                println("Zenn: $zenn")
            }) {
                Text("ポートフォリオ生成")
            }
        }
    }
}
```

## 型定義ファイルを作る

/src/main/kotlin/ProfileRequest.ktを作成し、以下記述。
APIがリクエストを受け取るとき、このクラスの形に自動でマッピングされるようにするためです。

```bash
package com.example  // main.ktと同じに合わせる

import kotlinx.serialization.Serializable

@Serializable
data class ProfileRequest(
    val name: String,
    val bio: String,
    val github: String,
    val zenn: String
)
```

## Routingの設定を作る

/src/main/kotlin/Routing.ktに、以下の記述をする。
```bash
package com.example

import io.ktor.server.application.*
import io.ktor.server.request.receive
import io.ktor.server.response.*
import io.ktor.server.routing.*

fun Application.configureRouting() {
    routing {
        post("/profile") {
            val request = call.receive<ProfileRequest>()

            println("受信: $request")

            call.respond(mapOf("status" to "ok"))
        }
    }
}
```

## フロント側のAPI連携コードを作る
