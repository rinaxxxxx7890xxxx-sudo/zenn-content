# AWSってなんやねん
では、そろそろ本題のAWSについて触れていきましょう。

### AWS - Amazon Web Services(アマゾン ウェブ サービス)の略
これは名前の通り、Amazon社が提供しているクラウドサービスです。

##### クラウドサービスって何？と思ったあなたへ
一言でいうと、「インターネット越しに、必要なときだけ必要な分、サーバーやネットワークをレンタルできる仕組み」のことです。

先ほど、「インフラ構築の手順」として以下の流れを説明しました。
* サーバーの設置
* OSのインストール
* ソフトウェアのインストール・設定

従来のインフラ構築（`オンプレミスと呼びます`）では、この「サーバーの設置」をするために、
自分で物理的なパソコン（サーバー機器）を買いに行き、自宅や会社の部屋に設置して、電源やネットのケーブルを挿すという作業が必要でした。

しかし、クラウドサービス（AWSなど）を使えば、貸主(Amazon)が巨大なデータセンターに用意してくれている大量のサーバーの中から、
「ちょっと1台貸して！」とボタンを数回ポチポチするだけで、ものの数分で自分専用のサーバーが手に入っちゃいます。
つまり、オンプレミスのように自分で色々買いに行ったり設置したりする手間がかかりません。

![イメージ画面11](/images/image11.png)

このようにネットワーク上で簡単にサーバーを手に入れることができちゃうサービスを、**クラウドサービス** と呼んでます。

### AWSのサービスについて
AWSには、細かな機能も含めると100以上のサービスが存在しており、クラウドサービスとしては世界最大規模です。

ちなみに、有名どころのクラウドサービスはAWS以外にも存在します。
* GCP（Google Cloud）
* Azure（Microsoft Azure）

などなど、他にもいくつか種類がありますので気になった方は調べてみてください。

### AWSの特徴について
1. サービスが非常に豊富
図でお分かりいただけたかもしれませんが、多すぎるくらいサービスが多いです。
2. リソースを柔軟に調達できる
必要な時に、必要な量だけ、瞬時にサーバーのリソース（容量）を増やしたり減らしたりできます。
3. 従量課金制度
レンタルサーバーやVPSと違い、月額料金ではなく、使った分を払えばいいシステムです。
なお、設定した予算を超えそうになったら通知する機能があるため、気づかないうちに使いすぎることは防げます。
